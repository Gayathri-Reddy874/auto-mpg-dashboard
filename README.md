# 📌 Project Overview

An industry-level **Data Analytics & Business Intelligence Dashboard Project** focused on vehicle fuel efficiency analysis using the famous **Auto MPG Dataset**.

This project combines:

* 📊 Exploratory Data Analysis (EDA)
* 📈 Interactive Business Intelligence Dashboards
* ⚡ Real-Time KPI Monitoring
* 🎯 Executive-Level Analytics
* 🌐 Interactive Web Applications using Streamlit & Dash

The project analyzes:

* Fuel efficiency trends
* Horsepower impact
* Weight vs MPG relationship
* Vehicle market distribution
* Performance analytics by region

# 🧠 Business Insights

* Fuel efficiency improved significantly over time.
* Japanese vehicles demonstrated superior MPG performance.
* Heavier vehicles negatively impact fuel efficiency.
* Higher horsepower generally reduces mileage.
* Smaller engines provide better fuel economy.
* Weight and horsepower strongly correlate with MPG performance.

# ▶️ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Gayathri-Reddy874/auto-mpg-dashboard.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd auto-mpg-dashboard
```

---

## 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

## 4️⃣ Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6️⃣ Run Streamlit Dashboard

```bash
streamlit run app.py
```

---

## 7️⃣ Run Dash Dashboard

```bash
python dash_app.py
```
